## Getting started
### Installation
  
```sh
# Install rush
npm install -g @microsoft/rush

# Install all dependencies
rush update

# Build all packages
rush build

# Run demos
# FixedLayout
rush dev:demo-fixed-layout
# FreeLayout
rush dev:demo-free-layout
```
